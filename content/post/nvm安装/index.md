## nvm下载地址
[下载地址](https://github.com/coreybutler/nvm-windows/releases)

选择nvm-setup.exe

选择目录和nodejs环境目录直接下载即可

查看版本
```
nvm version
```
## 安装nodejs

### 安装指定版本node
```
nvm install 16.18.0
```
### 切换指定版本node
```
nvm use 16.18.0
```
### 查看已安装的node列表
```
nvm list
```
### 卸载指定版本node
```
nvm uninstall 16.18.0
```
